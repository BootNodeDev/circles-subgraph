Circles subgraph for [The Graph](https://thegraph.com/) to fastly query available data in the Circles UBI ecosystem.

## Development

```
// Install dependencies
npm install

// Copy env file and edit it
cp .env.example .env

// Use graph commands
npm run codegen
npm run build
npm run create
npm run deploy
```
